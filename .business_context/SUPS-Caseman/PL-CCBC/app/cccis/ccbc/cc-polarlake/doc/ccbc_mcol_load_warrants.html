Details:

The circuit is designed to  take the data that has been inserted earlier in the morning in the Media Centre database and copy it across to the SUPS CCBC database. The data will be taken across exactly as it is,  certain sequence number will need to be regenerated in order to be relative to the SUPS CCBC database. Finally, the STATUS column of the TAPE_FILES table in the Media Centre database will be set to “T” to indicate that the data has been transferred to SUPS.
