Building and Deploying the Transformation Server
--------------------------------------------------------------------
To run this server in the same instance machine as the normal sups server. 

1. Ensure you have setup the transformation server under Jboss. Unzip the transformation.zip into Drive?:\tools\extracted\jboss-4.0.1\server\ .
2. Build the transformation.ear by calling ant from the root transformations server. This will deploy the ear into the JBOSS transformation server.
3. To start the transformation type the following run -c tranformation from the cmd promat in the folder structure Drive?::\tools\extracted\jboss-4.0.1\bin
