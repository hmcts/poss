Constructor Detail

WPXPath

function WPXPath(ref, path)
    Word Processing XPath class Facilitates easy access to configurable xpahts. The JavaScript WP code only uses references to XPaths. These references are static attributes to this class. The actual xpath corresponding to the reference is loaded from the controller configuration xml.
