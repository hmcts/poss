process

function process(WPS, formctrl, context, callback, alwaysCallback)
    SUPS Word Processing Controller Process function 
    Invoked by the WPCtrl (delegating the WP.Process(...) call.
 Subclasses MAY override this function - provided this implementation is also used.
    Parameters:
      context - xml fragment stating the context for the request
      callback - callback (function or formname)
      formController - formController managing the 'current' form requesting the word processing
      allwaysCallback - indicator whether or not to always call the callback, or only after the controller made the appliction navigate away from the 'current' form.
