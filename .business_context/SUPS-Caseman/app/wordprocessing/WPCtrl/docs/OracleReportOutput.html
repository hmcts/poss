Constructor Detail

OracleReportOutput

function OracleReportOutput(description, eventId, eventCode, refCJR, refN, orderID, mainOrderID, hearingType)
    The OracleReportOutput JavaScript Object represents 'an output'. A OracleReportOutput is identified by - ref_n: the N* reference, as the customer prefers refering to outputs - ref_cjr: the CJR* reference, as eds refers to outputs (since RFC ...) - ref_txt: output title, excluding the above refs. The OracleReportController uses the OracleReportOutput object instances to identify the output needing creation. (The WPController will pass an xml snippet to the WPOutput's selector function).
